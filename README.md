# flow-examples
Sample project demonstrating Flow typing in different contexts.

Each file in /src demonstrates a different concept.

To get code (npm):
```bash
git clone https://github.com/AlanMunson/flow-examples.git
cd flow-examples
npm i
npm run flow-check
```
If using Yarn:

```bash
git clone https://github.com/AlanMunson/flow-examples.git
cd flow-examples
yarn
yarn flow-check
```
